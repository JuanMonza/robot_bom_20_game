# robot_bom_20_game

**Juego Bomberman Jardines** - Versión moderna con temática de aviación

## Descripción

Juego estilo Bomberman desarrollado con HTML5 Canvas y JavaScript vanilla. Incluye personajes de aviación, enemigos monstruosos, aliados robóticos y efectos visuales realistas.

## Características

- **Personajes temáticos**: Azafata, Azafato y Piloto
- **Enemigos inteligentes**: Monstruos amarillos con IA estilo Pac-Man
- **Aliados**: Robot azul que ayuda desde el nivel 3
- **Bombas automáticas**: 30% de probabilidad al destruir bloques
- **Efectos realistas**: Explosiones circulares, bloques estilo ladrillo
- **Controles móviles**: Soporte para dispositivos táctiles
- **Cumplimiento legal**: Políticas de datos según legislación colombiana

## Tecnologías

- HTML5 Canvas
- JavaScript ES6+
- CSS3 (Glassmorphism)
- Web Audio API

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/JuanMonza/robot_bom_20_game.git
```

2. Abre `index.html` en tu navegador

## Cómo Jugar

- **Teclas de dirección**: Mover personaje
- **Barra espaciadora**: Colocar bomba
- **Objetivo**: Eliminar todos los enemigos sin morir

### Niveles

- **Nivel 1**: 4 enemigos
- **Nivel 2**: 6 enemigos
- **Nivel 3+**: 8+ enemigos + Robot aliado azul
